About

The library currently supports the following services:

  1. Classify: classify books, videos, CDs and other materials based on identifier: ISBN, OCLC Number, UPC, ISSN, title, author, FAST headings, or FRBR work identifier.
  2. xID API: submit an identifer such as an ISBN, ISSN, or OCLC Number to this service, and it returns a list of related identifiers and selected metadata.
  3. WorldCat Recommender Service: provides recommendations for books and other materials based on data in WorldCat.

I hope to include the following services in future updates:

  1. QuestionPoint Knowledge Base API: access to QuestionPoint knowledge base Question and Answer (Q&A) pairs that QuestionPoint libraries saved for future re-use, for FAQ pages, for patron searches. The QuestionPoint global knowledge base is a collection of Q&A pairs contributed by nearly 500 libraries worldwide. Local knowledge bases are built and maintained by single libraries or regional library groups and usually contain information more specific to that group.
  2. WorldCat knowledge base API: read-only service for e-resource discovery and linking. Provides developer-level access to a library’s information in the WorldCat knowledge base. The WorldCat knowledge base combines data about your library’s econtent with access to it through linking features.
  3. WorldCat Registry: access to information in the Worldcat Registry—a free global directory for libraries, consortia, archives and museums.
  4. WorldCat Search API: access to WorldCat for bibliographic holdings and location data.
